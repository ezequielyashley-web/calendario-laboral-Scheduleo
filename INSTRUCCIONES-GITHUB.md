## 🚀 **PASOS PARA SUBIR A GITHUB**

### **1. Descomprimir el ZIP**
```bash
# Descomprimir en una carpeta
# Te creará: scheduleo-nextjs-completo/
```

### **2. Ir a la carpeta**
```bash
cd scheduleo-nextjs-completo
```

### **3. Probar localmente (OPCIONAL)**
```bash
# Instalar dependencias
npm install

# Ejecutar
npm run dev

# Abrir http://localhost:3000
```

### **4. Subir a GitHub**

**Tu repositorio ya configurado:**
`https://github.com/ezequielyashley-web/calendario-laboral-Scheduleo`

```bash
# Inicializar Git
git init

# Añadir todos los archivos
git add .

# Commit
git commit -m "feat: scheduleo desktop v2.0 completo - 10 fases"

# Renombrar rama a main
git branch -M main

# Conectar con GitHub
git remote add origin https://github.com/ezequielyashley-web/calendario-laboral-Scheduleo.git

# Subir
git push -u origin main
```

**Si el repo ya existe y da error:**
```bash
git push -f origin main
```
